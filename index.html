Tutorial de Async/Await en JavaScript

¿Qué es Async/Await?

Async/Await es una forma de escribir código asíncrono en JavaScript que lo hace parecer y comportarse como código síncrono. Esto hace que el código sea más fácil de leer y entender.

Async

La palabra clave async se utiliza para declarar una función asíncrona. Una función async siempre devuelve una promesa.

async function miFuncionAsincrona() {
    return "Hola, soy una función asíncrona";
}

// Uso de la función asíncrona
miFuncionAsincrona().then(console.log);

Await

La palabra clave await se utiliza dentro de una función async para esperar a que se resuelva una promesa. Hace que JavaScript espere hasta que la promesa se resuelva y devuelva su resultado.

async function obtenerDatos() {
    let respuesta = await fetch('https://api.ejemplo.com/datos');
    let datos = await respuesta.json();
    return datos;
}

// Uso de la función
async function mostrarDatos() {
    let datos = await obtenerDatos();
    console.log(datos);
}

Manejo de errores

Puedes usar try/catch para manejar errores en funciones async/await:

async function manejarErrores() {
    try {
        let datos = await obtenerDatos();
        console.log(datos);
    } catch (error) {
        console.error("Hubo un error:", error);
    }
}

Conclusión

Async/Await hace que el código asíncrono sea más fácil de escribir y entender. Permite escribir código asíncrono que se parece y se comporta como código síncrono, lo que facilita el manejo de operaciones asíncronas complejas.
